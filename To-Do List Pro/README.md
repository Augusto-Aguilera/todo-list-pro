# 📝 To-Do List Pro

Aplicación de gestión de tareas desarrollada con JavaScript puro.

## 🚀 Demo
Permite crear, completar y eliminar tareas con persistencia en el navegador.

## ✨ Funcionalidades

- Crear tareas
- Marcar como completadas
- Eliminar tareas
- Filtros (todas / pendientes / completadas)
- Contador de tareas
- Persistencia con localStorage

## 🛠️ Tecnologías

- HTML5
- CSS3
- JavaScript

## 🎯 Objetivo

Este proyecto fue desarrollado para practicar lógica de programación, manipulación del DOM y manejo de estado en aplicaciones frontend.
