# 📝 To-Do List Pro | Gestión Eficiente de Tareas

Una aplicación de productividad robusta desarrollada íntegramente con **JavaScript Vanilla**. Este proyecto demuestra mi capacidad para gestionar el estado de una aplicación sin depender de frameworks externos.

---

## 🚀 Demo en Vivo
[Mira la aplicación funcionando aquí](https://augusto-aguilera.github.io/todo-list-pro/)

---

## ✨ Funcionalidades Destacadas
- **CRUD de Tareas:** Crear, Leer, Actualizar (marcar como completadas) y Eliminar tareas de forma intuitiva.
- **Filtros Inteligentes:** Clasificación dinámica por estado (*Todas / Pendientes / Completadas*).
- **Persistencia de Datos:** Integración con `localStorage` para que tus tareas no se borren al recargar la página.
- **Contador Dinámico:** Visualización en tiempo real del total de tareas pendientes.
- **Diseño Responsive:** Interfaz optimizada para móviles y escritorio.

---

## 🛠️ Stack Tecnológico
- **HTML5:** Estructura semántica para accesibilidad.
- **CSS3:** Diseño moderno con Flexbox y variables para una interfaz limpia.
- **JavaScript (ES6+):** - Manipulación avanzada del **DOM**.
  - Manejo de **Event Listeners** delegados.
  - Uso de **JSON** y **Web Storage API** para la persistencia.

---

## 🎯 Desafío Técnico
El mayor aprendizaje en este proyecto fue la implementación de la **persistencia de datos**. Logré que la aplicación sea funcional incluso después de cerrar el navegador, transformando objetos de JavaScript a cadenas JSON y viceversa para interactuar correctamente con el almacenamiento local.
